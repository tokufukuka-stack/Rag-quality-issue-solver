# RAG 检索质量问题解决方案库

本仓库收录了 RAG（Retrieval-Augmented Generation）系统中常见问题的解决方案，涵盖检索质量、文档处理、生成质量、系统架构、安全隐私、评估优化六大类。

## 问题目录

完整问题清单见 [RAG检索质量问题.txt](RAG检索质量问题.txt)，包含以下六个方向：

1. **检索质量问题** — 语义漂移、关键词依赖、长尾召回、跨语言检索等
2. **文档处理与索引问题** — PDF 解析、Chunk 切分、元数据保留、增量更新等
3. **生成质量问题** — 幻觉、上下文溢出、引用溯源、推理链断裂等
4. **系统架构与工程问题** — 延迟优化、扩展性、缓存策略、多轮对话等
5. **安全与隐私问题** — 提示注入、数据泄露、权限控制、知识投毒等
6. **评估与优化问题** — 统一指标、评估集构建、超参数调优、回归测试等

每个问题对应一个独立目录，包含：
- `解决方案.md` — 具体解决方案
- `可行性分析.md` — 方案的可行性评估
